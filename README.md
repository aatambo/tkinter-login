# Portfolio-Django

Login system with tkinter and sqlalchemy

## features

users sign up,sign in and sign out

## Installation

Using the package manager [pip]

```bash
pip install -r requirements.txt
```

## Usage

Run on your terminal the following

```bash
python auth.py
```
